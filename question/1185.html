灰原哀粉籍测试题

一 基础知识 这些是一些关于灰原哀的基础性的选择题

1.  日本动漫《名侦探柯南》中的主要人物，本名宫野志保 。原为黑衣组织的成员雪莉（Sherry），是能使身体缩小的药物——“APTX4869”的发明者。在姐姐宫野明美遭组织杀害后，因反抗组织而被囚禁，服下APTX4869，身体缩小后背叛组织逃亡，化名灰原哀，寄宿于阿笠博士家中，就读于帝丹小学哪个班级??
  • A 一年A班
  • B 一年B班
  • C 一年C班
  • D 一年D班
解析  常识题
2.  灰原哀的父亲是日本人，他的母亲是哪国人
  • A 法国人
  • B 德国人
  • C 英国人
  • D 美国人
解析  灰原哀的母亲是英国人
3.  下列哪个是灰原哀喜欢的颜色？
  • A 绯色
  • B 红色
  • C 白色
  • D 紫色
解析  TV 176、178
4.  灰原也有使用女性时尚用品的爱好，请问灰原哀喜欢下列哪个品牌的钱包
  • A 路易·威登
  • B 安娜苏
  • C 古驰
  • D 芙莎绘
解析  TV 375中提出的帮柯南的条件：“芙莎绘最新款的钱包，这么大……（用手比划大小）就这样我就帮你。”
5.  下列哪个人物不知道灰原哀的真实身份
  • A 库拉索
  • B 安室透
  • C 皮斯克
  • D 贝尔摩得
解析  皮斯克曾见过幼时志保，没有来得及报告组织便被杀（TV 177），贝尔摩得第一次见到灰原便确定其身份（TV 339），具体原因不详，可能是她见过幼年志保。对组织隐瞒了相关情况。库拉索恢复记忆后通过对话猜出灰原哀的真实身份是逃出组织的雪莉（M 20《纯黑的噩梦》）

请在这里填写你的答案：
